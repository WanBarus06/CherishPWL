##Installation
- Laravel Breeze
- Livewire

## Database
- cherish

## Features
- Login : Masuk ke dalam website sebagai user (breeze)
- Register : Mendaftar agar dapat menjadi user (breeze)
- Verify Email : Memverifikasi email user yang telah mendaftar (breeze)
- Forgot Password : Mengganti password user via email (breeze)
- Change Password : Mengganti password user dari dalam website (breeze)
- Logout : Mengancurkan session untuk membuat user dapat login kembali (breeze)
- Validation Form : Membuat validasi di form seperti login, register, change password (breeze)
- Pagination : Membatasi halaman random article di all article dan single article(livewire)
- Save Article : Menyimpan halaman artikel (livewire)
- Upload Profile Picture : Menyimpan gambar profil dari user yang sudah terautentikasi (livewire)
- Searching : Mencari artikel yang akan memunculkan judul yang dicari (livewire)
- Add Comment : Menambahkan komentar di dalam tiap single artikel
- Reply Comment : Membalas komentar dari komentar parent yang ada dalam bagian komentar
- Authorization : Membedakan level role admin dan user dalam website dan membatasi hak akses ke halaman admin
- CRUD : Fitur membuat melihat dan menghapus komentar dan user serta mengedit artikel dengan tinyMCE(wysiwyg)
- Change Profile : mengganti informasi dari user seperti first name last name dan additional data
